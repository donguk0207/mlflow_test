MLFlow를 실습하기 위한 단계
- Data Preprocessing
  -> 모델 최적화를 위한 데이터 전처리 진행
- Model 학습과 성능 개선
  -> 모델학습 진행
  -> MLflow를 활용해서, 모델 학습결과를 모니터링 할 수 있도록 코드 구현
  -> MLflow를 활용해서, HyperParameter tuning -> 결과를 모니터링 할 수 있도록 구현
- Model Selection & Model Registration
  -> MLflow를 활용해서, 여러 모델의 결과를 조회하고, 모델 선택
  -> MLflow를 활용해서, 선택한 모델을 Registration -> 관리 진행
- Model Serving(Simple Test)
  -> Registration된 모델을 Load해서 추론에 사용
  -> MLflow를 활용해서, 해당 모델을 REST API 형태로 Serving 함.
